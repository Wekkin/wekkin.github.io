PyCharm配置

Posted on 2023-12-19 Edited on 2024-01-03

一般情况下，代码文件的开头会包含一些注释，用于说明文件的信息，比如作者名字、创建时间、文件类型等。这种信息的格式可能会因组织或项目而异。
在PyCharm中，你可以配置代码模板，让它在创建新文件时自动包含你想要的注释信息。以下是一种设置方式：

  1. 打开PyCharm，点击顶部菜单的 File（文件） -> Settings（设置）。

  2. 在设置窗口中，选择 Editor（编辑器） -> File and Code Templates（文件和代码模板）。

  3. 在 Files（文件）选项卡下，选择你想要编辑的文件类型（例如Python文件）。

  4. 在模板编辑框中，你可以编写文件的模板，比如：

    在这个模板中，${DATE} 和 ${TIME} 是PyCharm的模板变量，它们会在文件创建时自动替换为当前日期和时间。你也可以添加其他信息或变量，根据需要进行自定义。

  5. 编辑完毕后，点击 OK 保存设置。
